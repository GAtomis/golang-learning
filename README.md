## 学习GO 

 * build:编译包和依赖
* clean:移除对象文件
* doc:显示包或者符号文档
* env: 打印go的环境信息
* bug:启动错误报告
* get:下载包
* install:安装源码包

### 使用go get
初始化
```
//初始化
go mod init started
```
安装remote
```
//初始化
go get github.com/go-sql-driver/mysql
```
